# AgenticApplications

This repository now contains two separate demos:

- `simple/`: single-agent calculator demo
- `medical/`: combined-server medical multi-agent demo

Each folder is self-contained with its own:

- source files
- `requirements.txt`
- `Dockerfile`
- README with run instructions

## Quick start

Simple demo:

```bash
cd simple
pip install -r requirements.txt
python agent.py
```

Medical demo:

```bash
cd medical
pip install -r requirements.txt
python medical_combined_server.py
```
